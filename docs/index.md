# Job Listing — DevOps Pipeline

12-week project building a production-grade DevOps pipeline for a job listing application.

## Stack

| Layer | Technology |
|-------|------------|
| Application | Go 1.21 |
| Containerization | Docker, Docker Compose |
| Cloud | AWS (EC2, EKS, RDS) |
| CI/CD | GitHub Actions |
| IaC | Terraform |
| Monitoring | Prometheus, Grafana |
| GitOps | ArgoCD |

## Project Phases

- **Phase 1** — CI/CD Foundation (Weeks 1-4)
- **Phase 2** — Infrastructure (Weeks 5-8)
- **Phase 3** — Production Operations (Weeks 9-12)
